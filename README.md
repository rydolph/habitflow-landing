# HabitFlow Landing Page

**HabitFlow** — это стильная и современная лендинг-страница, предназначенная для демонстрации продукта. Она включает в себя все необходимые страницы для представления информации о продукте, а также интерактивные элементы для удобства пользователей.

## Состав проекта

- `index.html` — главная страница проекта.
- `404.html` — страница ошибки 404 (не найдено).
- `contacts.html` — страница с контактной информацией.
- `faq.html` — страница с часто задаваемыми вопросами (FAQ).
- `assets/` — папка с изображениями и другими ресурсами для сайта.

## Использование

- Онлайн версия доступна по ссылке: [ваш сайт](https://rydolph.github.io/habitflow-landing/).
